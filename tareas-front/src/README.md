# 📝 App de Tareas (Fullstack)

Mini aplicación fullstack para gestionar tareas.

## 🚀 Tecnologías
- Backend: FastAPI (Python)
- Frontend: React + Vite
- Persistencia: JSON

## ⚙️ Cómo correr el proyecto

### 1. Backend
```bash
cd python-proyectos
py -m uvicorn app:app --reload

### 2. Frontend
cd python-proyectos/tareas-front
npm install
npm run dev
